## 기여하기

- 대상: 해양대 학생이라면 누구나

## 주요 기능
한국해양대 학생들을 위해 필요한 웹사이트를 모아놓은 익스텐션입니다.

총 6개의 사이트를 제공하고 있으며 내용은 아래와 같습니다.
- 한국해양대 홈페이지
- 해양대 도서관
- 해양대 LMS
- 수강신청
- 비교과 활동
- 해양대 에타
